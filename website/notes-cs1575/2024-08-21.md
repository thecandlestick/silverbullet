## Loops

[[examples/cpp-loops]]
while loops
```c++
#include <iostream>
using namespace std;

int main()
{
  int n = 10;

  while( n>0 )  // check condition
  {
    cout << n << ", ";  // execute body
    --n;
  }  // repeat

  cout << "liftoff! \n"
```


## Scope

[[examples/cpp-scope]]
```c++
// "Global" Scope
#include <iostream>
using namespace std;

int main() 
{
  // "outer" scope
  int x = 10;
  int y = 20;
  {
    // {} defines "inner" scope
    int x;
    ::x = 20;
    x = 50;   //sets value for (inner) x
    y = 50;   //sets value for (outer) y
    cout << "inner block:\n"
         << "x: " << x << '\n'  // x: 50
         << "y: " << y << '\n'; // y: 50
  }
  cout << "outer block:\n"
       << "x: " << x << '\n'   // x: 10
       << "y: " << y << '\n';  // y: 50

  return 0;
}
```

## Classes

[[examples/cpp-classes]]
A typical class
```c++
class name_of_type
{
  public:
    // function prototypes here
  private:
    // member data here
};
```

```c++
#include <iostream>
using namespace std;

class Rectangle
{
  private:
    int width, height;
  public:
    void set_values(int, int);
    //You don't have to give names to parameters in prototypes! ðŸ¤¯
    int area() {return width*height;} //"inline" function
};

int main()
{
  Rectangle rect, rectb;
  rect.set_values(3, 4);
  rectb.set_values(5, 6);
  cout << "rect area: " << rect.area() << endl
       << "rectb area: " << rectb.area() << endl;
  return 0;
}

// <Return Type> <Namespace(class name)>::<Function>(<Parameters>)
void Rectangle::set_values(int x, int y)
{
  width = x;
  height = y;
}
```


constant functions
```c++
int Rectangle::area();
//vs.
int Rectangle::area() const; // area() cannot alter width/height

void Rectangle::set_values(int x, int y);
//vs.
void Rectangle::set_values(int x, int y) const; // x/y can be altered
```

## Templates

[[examples/cpp-templates]]
```c++
#include <iostream>
using namespace std;

template <typename T>
T sum(T a, T b)
{
  T result;
  result = a + b;
  return result;
}

int main()
{
  int i=5, j=6, k;
  double f=2.0, g=0.5, h;
  k=sum<int>(i, j);
  h=sum<double>(f, g);

  cout << k << endl;
  cout << h << endl;
  return 0;
}
```
